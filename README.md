# 🪭 TonfernPDF v4.0 - เครื่องมือจัดการ PDF ระดับมืออาชีพ (Local-First)

**TonfernPDF** คือเว็บแอปพลิเคชันระดับพรีเมียมสำหรับการจัดการไฟล์ PDF แบบท้องถิ่น (Local-first) ที่ออกแบบมาเพื่อประสิทธิภาพสูงสุดโดยไม่ทิ้งเรื่องความปลอดภัยของข้อมูล มั่นใจได้ว่าไฟล์ของคุณ **จะไม่ถูกอัปโหลดขึ้นเซิร์ฟเวอร์ใดๆ** และประมวลผลภายในเครื่องของคุณ 100%

## 🌟 ทำไมต้อง TonfernPDF?

เครื่องมือ PDF ออนไลน์ส่วนใหญ่ต้องการให้คุณอัปโหลดเอกสารไปยังเซิร์ฟเวอร์ของพวกเขา แต่ **TonfernPDF** ต่างออกไป เราใช้เทคโนโลยีการประมวลผลฝั่งไคลเอนต์ที่ทรงพลังเพื่อจัดการทุกอย่างในเบราว์เซอร์ของคุณโดยตรง

- **🔒 ความเป็นส่วนตัวสูงสุด**: ไม่มีการอัปโหลดไฟล์ ข้อมูลของคุณยังคงอยู่ในเครื่อง 100%
- **🚀 มาตรฐาน Enterprise**: อินเทอร์เฟซที่ลื่นไหล ตอบสนองไว พร้อมการทำงานที่แม่นยำระดับมืออาชีพ
- **🧭 ประสบการณ์อัจฉริยะ**: ระบบคัดกรองเครื่องมือตามกลุ่มผู้ใช้งาน (Persona) และแถบแสดงสถานะแบบ Real-time
- **🎨 ดีไซน์พรีเมียม**: รูปลักษณ์ทันสมัย พร้อมธีมสีมงคล (ทอง-แดง) ที่ดูหรูหราและใช้งานง่าย

## 🛠️ ฟีเจอร์หลัก

### 📂 การจัดการเอกสาร
*   **รวมและแยกไฟล์ (Merge & Split)**: รวมเอกสารหลายไฟล์เข้าด้วยกัน หรือแยกเฉพาะหน้าที่ต้องการออกมาเป็นไฟล์ใหม่หรือไฟล์ ZIP
*   **จัดเรียงหน้า (Organize Pages)**: อินเทอร์เฟซแบบลากวาง (Drag-and-drop) เพื่อสลับลำดับหน้ากระดาษได้อย่างอิสระ
*   **ลบหน้า (Delete Pages)**: ลบหน้ากระดาษที่ไม่ต้องการออกได้อย่างง่ายดาย
*   **ลายน้ำ (Watermark)**: เพิ่มลายน้ำข้อความที่มีความแม่นยำสูง เลือกตำแหน่ง ปรับความใส และหมุนองศาได้ตามต้องการ

### ✍️ การเซ็นชื่อและกรอกฟอร์ม
*   **เซ็นชื่อ (Sign PDF)**: เครื่องมือเซ็นชื่อระดับโปร พร้อมระบบ **Magnetic Snapping (เส้นไกด์จัดกึ่งกลาง)** และการปรับความโปร่งใส (Opacity)
*   **เพิ่มข้อความ (Add Text)**: กรอกแบบฟอร์มหรือเพิ่มคำอธิบายด้วยขนาดตัวอักษรที่ปรับเปลี่ยนได้ (10pt - 32pt)

### 🗜️ การเพิ่มประสิทธิภาพและการแปลงไฟล์
*   **การบีบอัดขั้นสูง (Advanced Compression)**:
    *   *Mode A*: เพิ่มประสิทธิภาพโครงสร้างไฟล์และ Metadata
    *   *Mode B (Aggressive)*: แปลงหน้าเป็นรูปภาพใหม่เพื่อลดขนาดไฟล์อย่างมหาศาล (เหมาะสำหรับเอกสารสแกน)
*   **PDF ↔ รูปภาพ**: แปลงไฟล์ระหว่าง PDF และรูปภาพคุณภาพสูง (JPG, PNG, WebP)
*   **ดึงข้อมูลเสริม**: ดึงรูปภาพต้นฉบับที่ฝังอยู่ในไฟล์ หรือดึงเนื้อหาข้อความออกมาใช้งานต่อ

### 🔒 ความปลอดภัย
*   **ป้องกันไฟล์ (Protect PDF)**: เพิ่มรหัสผ่านที่แข็งแกร่งและระบุสิทธิ์การเข้าถึง (การพิมพ์, การคัดลอก)
*   **ปลดล็อกไฟล์ (Unlock PDF)**: ถอนรหัสผ่านออกจากไฟล์ที่ได้รับการป้องกัน (ต้องทราบรหัสผ่านเดิม)

## 🧭 ระบบ Workflows ตามกลุ่มผู้ใช้งาน (Persona)
หน้าจอจะปรับโหมดการแสดงผลให้เหมาะสมกับบทบาทของคุณเพื่อให้เข้าถึงเครื่องมือที่จำเป็นได้ไวที่สุด:
- 🎓 **นักเรียน (Student)**: เน้นการ Merge, Split และการแปลงรูปภาพเป็น PDF
- 💼 **พนักงานออฟฟิศ (Office)**: เน้นการบีบอัดไฟล์, การเซ็นชื่อ และการจัดเรียงหน้า
- ⚖️ **กฎหมาย (Legal)**: เน้นเรื่องความปลอดภัย, การเซ็นชื่อ และการใส่ลายน้ำ
- 🛠️ **วิศวกร (Engineer)**: เน้นการดึงรูปภาพละเอียดสูงและการบีบอัดไฟล์ขั้นสูง

## 💻 เทคโนโลยีที่ใช้

TonfernPDF ถูกพัฒนาด้วยมาตรฐานเว็บแอปพลิเคชันยุคใหม่:
- **Core**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+).
- **PDF Logic**: [pdf-lib](https://github.com/Hopding/pdf-lib), [pdf.js](https://github.com/mozilla/pdf.js).
- **Imaging**: [jsPDF](https://github.com/parallax/jsPDF).
- **Utility**: [JSZip](https://stuk.github.io/jszip/) สำหรับการส่งออกไฟล์จำนวนมาก, [SortableJS](https://sortablejs.github.io/Sortable/) สำหรับการลากวาง

## 🚀 เริ่มต้นใช้งาน

เนื่องจาก TonfernPDF เป็นเครื่องมือแบบ Standalone คุณสามารถเปิดใช้งานได้ทันทีผ่านเว็บเบราว์เซอร์:

1.  Clone repository นี้:
    ```bash
    git clone https://github.com/Earthondev/Tonfernpdf-.git
    ```
2.  เปิดไฟล์ `Tonfernpdf.html` ด้วย Chrome, Firefox หรือ Safari
3.  เริ่มจัดการไฟล์ PDF ของคุณด้วยความสบายใจสูงสุด
